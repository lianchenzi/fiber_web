## Build Setup

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:8082
yarn run dev

# build for production with minification
yarn run build

# visualize output files
yarn run analysis

# copy output files to docs folder
yarn run copy

# preview output files
yarn add http-server -g
cd dist
http-server
```
